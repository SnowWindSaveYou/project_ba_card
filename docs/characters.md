# 《轻拳飞扬》角色人设文档

> 参考风格：碧蓝档案
> 版本：v0.2
> 最后更新：2026-05

---

## 首发四名角色总览

| 角色 | 流派 | 身高 | 发色 | 气质 |
|------|------|------|------|------|
| 一之濑枫 | 剑道 | — | 深冬樱色（暗藕粉） | 冷静克制、正统贵气 |
| 夏琳 | 跆拳道 | — | 橙金色 | 活泼好动、爱出风头 |
| 云柔 | 太极 | 最高 | 纯白色（银光） | 温柔沉稳、从容压迫 |

## 一之濑枫（剑道）

**定位**：攻防均衡的正统派，架势（竹刀）连击

**气质**：冷静、克制、有贵气，像道馆里最认真的那个人

### 外貌

| 项目 | 设计 |
|------|------|
| 发色 | 黑色，内侧挑染红色（拨开外层头发才能看见，出招时随发丝飘动闪现） |
| 发型 | 黑长直，额前齐刘海；平时自然垂落，战斗时用枫叶发饰束起侧发 |
| 发饰 | 枫叶造型发饰（深红色，金属质感），固定在左侧耳后，是她的标志性单品 |
| 瞳色 | 钢铁蓝，中央有淡金色光圈 |
| 身材 | 修长，姿态端正，肩膀放松但背脊笔直 |

### 默认服装（对战服）

品牌：**桜風 (ŌKAZE)** — 和风学院风，深蓝白配色，樱花刺绣元素

| 部位 | 服装 | 说明 |
|------|------|------|
| 上半身 | 桜風·改良水手服 | 深蓝色水手领上衣，袖口樱花刺绣，胸前蝴蝶结 |
| 下半身 | 桜風·百褶短裙 | 深蓝百褶裙配白色运动内衬，裙摆有樱花暗纹 |
| 脚部 | 白色分趾足袋，无鞋 | 符合剑道实战感 |
| 护腕 | 传统棉布护腕（右手） | 标志性细节 |

**BA 风格元素**：水手领内侧有极细的金线暗纹，护具受损后金线断裂、发光颗粒掉落

**标志性细节**：耳后别着一根备用白色发带

## 夏琳（跆拳道）

**定位**：连招王，Combo 序列丰富，腿法华丽

**气质**：活泼好动、喜欢出风头，打架时爱笑，比赛前还会撩拨对手

### 外貌

| 项目 | 设计 |
|------|------|
| 发色 | 栗茶色，阳光下带暖橙光泽 |
| 发型 | 双侧马尾，位置偏低偏后（参考奈叶），用白色宽丝带扎起，丝带末端有蝴蝶结；发尾微微内卷 |
| 瞳色 | 明亮的蓝色，眼神直率，眼角微微上扬，笑起来会眯成月牙 |
| 身材 | 偏修长的运动型，腿部线条格外突出 |

### 默认服装（对战服）

品牌：**DASH** — 韩系运动潮牌，白底撞色，机能面料

| 部位 | 服装 | 说明 |
|------|------|------|
| 上半身 | DASH·露脐运动衫 | 白底蓝色收边短款速干衫，背后有 DASH 大 logo |
| 下半身 | 白色竞技裤 | 裤侧一道细蓝线，光脚 |
| 发饰 | 白色宽丝带蝴蝶结 | 扎双侧马尾用，战斗时被冲击波吹散飞扬 |

> DASH 暂无专属下半身护具，竞技裤为基础对战装

**BA 风格元素**：白色丝带在战斗激烈时被汗水打湿、微微透明，有一种不经意的少女感

**标志性细节**：左脚踝绑着蓝色细布带（旧款，来自启蒙教练），护甲胸口有她自己绣上去的蓝色小星星徽章

## 云柔（太极）

**定位**：厚血厚防，反击制裁，越打越稳

**气质**：温柔沉稳，说话轻声细语，打起来有一种让人窒息的从容——她不急，她等你犯错

### 外貌

| 项目 | 设计 |
|------|------|
| 发色 | 纯白色，质感如丝绸，阳光下泛出极淡的银光 |
| 发型 | 长发，平时用白玉簪盘起；出招时会半散开，散发能到腰部 |
| 瞳色 | 淡墨绿色，平静如水面，眼神不凌厉，但对视时有无形压力 |
| 身材 | 中等身高，体型温柔，但站姿有一种根植于地面的稳感 |

### 默认服装（对战服）

品牌：**云裳** — 新中式国风，素雅配色，飘逸丝绸

| 部位 | 服装 | 说明 |
|------|------|------|
| 上半身 | 云裳·盘扣对襟衫 | 月白色真丝对襟上衣，云纹盘扣，袖口渐变水墨 |
| 下半身 | 练功灯笼裤 | 烟灰色宽松灯笼裤，透气轻便 |
| 脚部 | 黑面白底布鞋 | 太极实战标配 |
| 发饰 | 白玉簪 | 盘起长发，护具受损时浮现蓝色太极纹路 |

> 云裳暂无专属下半身护具，灯笼裤为基础对战装

**BA 风格元素**：白玉簪是发光道具——护具受损时簪上浮现蓝色太极纹路，功力可视化

**标志性细节**：左腕木珠手链（师父所赠），每次发动反击前她会甩袖，手链微微闪动

## 铁拳小桃（拳击）

**定位**：高爆发高风险，震慑压迫，一拳定胜负

**气质**：娇小但气场不小，站在对手面前要仰头看人，但毫不怵场。嗓门大，喜欢喊口号，被打趴了还在笑着说"再来"

### 外貌

| 项目 | 设计 |
|------|------|
| 发色 | 粉红色 |
| 发型 | 短短的双马尾，扎得很高；刘海厚而直，盖到眉毛上方；打起来两个小揪揪跟着甩 |
| 瞳色 | 玫红色，眼睛相对脸偏大，表情变化极丰富 |
| 身材 | 约 148cm，四人中最矮；娇小纤细，但手腕手背有细微拳击肌肉线条；戴上绷带和手套后拳头看起来几乎和头一样大 |

### 默认服装（对战服）

品牌：**K.O.** — 美式街头朋克，黑红配色，铆钉皮革

| 部位 | 服装 | 说明 |
|------|------|------|
| 上半身 | K.O.·绑带运动内衣 | 红黑撞色运动束胸，交叉绑带设计 |
| 下半身 | K.O.·破洞牛仔热裤 | 做旧水洗热裤，膝盖铆钉护膝，链条腰饰 |
| 脚部 | 红色拳击鞋 | 厚底，微增高，没什么用 |
| 双手 | 白色绷带 + 红色拳击手套 | 绷带上歪歪扭扭手写"MOMO" |

**BA 风格元素**：护具破损时绷带局部撕裂扬起，发光粒子从拳头处爆散

**标志性细节**：耳朵上各一只小桃子形状耳钉——是她最女性化的特征，与整体形成反差萌

### 反差设计要点

娇小 + 拳击的反差在演出层面放大：

| 场景 | 演出方式 |
|------|---------|
| 出招动画 | 起手先蹲低蓄力，出拳瞬间腾空，体重全压在拳头上，冲击波比她人还大 |
| 华丽爆衣演出 | 冲击波把她整个人推开几步，站稳后粉红色爆散粒子 |
| 胜利动画 | 踩着对手倒地位置踮脚尖庆祝，还不够高，干脆爬上去 |
| 被打倒动画 | 倒了一下立刻弹起来拍拍裤子，比任何人都先站好 |
| 对阵云柔开场 | 小桃要仰头才能看到云柔的脸，但她先开口："你好高啊，等等我打你腿" |

## 视觉一致性原则

| 方向 | 说明 |
|------|------|
| 护具即角色"第二皮肤" | 每件护具能认出是谁的，与角色服装是同一套美术语言 |
| 爆衣后是另一种美 | 战损状态的战意可视化，参考碧蓝档案战斗立绘差分 |
| 发色与格斗气质对应 | 枫→克制暗藕粉，夏琳→活力橙金，云柔→沉稳黑蓝，小桃→爆发粉红 |
| 脚部设计各不相同 | 剑道足袋 / 跆拳道光脚 / 太极布鞋 / 拳击鞋，细节辨识度高 |

# 《血肉之战》(Flesh and Blood) 英雄与卡表参考

---

## 一、职业体系 (Classes)

FaB 共有 **10+ 个职业**，每个职业有独特的战斗风格：

| 职业 | 英文 | 战斗风格 | 代表机制 |
|------|------|---------|---------|
| **战士** | Warrior | 平衡攻防，武器连击 | Reprise（回击）、双剑流 |
| **忍者** | Ninja | 敏捷连击，Combo 链 | Combo（连击序列） |
| **蛮兽** | Brute | 高攻高风险，弃牌驱动 | 随机弃牌换取强力效果 |
| **守护者** | Guardian | 防御为主，厚甲 | Crush（粉碎，高伤触发） |
| **游侠** | Ranger | 远程射箭，军械库策略 | Arrow 子类型、Reload |
| **机械师** | Mechanologist | 科技道具，Boost 加速 | Boost（加速）、Item（道具链） |
| **法师** | Wizard | 奥术伤害，难以防御 | Arcane Damage（奥术伤害） |
| **刺客** | Assassin | 爆发伤害，Contract | Contract（合约暗杀） |
| **符文剑士** | Runeblade | 物理+奥术双重伤害 | Runechant Token（符文标记） |
| **幻术师** | Illusionist | 幻象攻击，虚实结合 | Phantasm（幻象，可被识破） |

### 天赋系统 (Talents)

部分英雄还拥有天赋，进一步定义卡池：

| 天赋 | 英文 | 特点 |
|------|------|------|
| **光明** | Light | 灵魂系列，治疗与净化 |
| **暗影** | Shadow | 血债、放逐区互动 |
| **元素** | Elemental | 冰/雷/大地融合 |
| **龙族** | Draconic | 龙火、龙的力量 |
| **皇家** | Royal | 宫廷权谋 |
| **神秘** | Mystic | 神秘学，预知 |

## 五、卡牌结构参考

### 5.1 典型攻击行动卡

```
┌────────────────────────────────────────┐
│  [●●●]                        [2]     │  ← Pitch 3 (蓝) / 费用 2
│                                        │
│        卡牌名称                         │
│        ─────────                        │
│   类型: Warrior Attack Action          │
│                                        │
│   效果文字描述                           │
│   "Go Again"                           │
│                                        │
│  [4]                           [3]     │  ← 攻击力 4 / 防御值 3
└────────────────────────────────────────┘
```

### 5.2 同名不同色示例

以 "Wounding Blow" 为例，同一张牌有三个版本：

| 版本 | Pitch 值 | 费用 | 攻击力 | 防御力 | 特点 |
|------|----------|------|--------|--------|------|
| 红色版 | 1 资源 | 0 | 6 | 3 | 攻击力最高 |
| 黄色版 | 2 资源 | 0 | 5 | 3 | 中庸 |
| 蓝色版 | 3 资源 | 0 | 4 | 3 | Pitch 值最高 |

→ 三个版本在构筑中被视为**不同的牌**，可以各带 3 张（经典构筑）或各 1 张（闪电战）

## 九、电子版开发参考数据

### 9.1 适合电子版简化的初始英雄推荐

建议从以下 4 个核心职业起步，每个职业选 1 位英雄：

| 职业 | 推荐英雄 | 理由 |
|------|---------|------|
| **战士** | Dorinthea Ironsong | 机制简洁，武器连击直观 |
| **忍者** | Katsu, the Wanderer | Combo 机制有趣，易于理解 |
| **守护者** | Bravo, Showstopper | 攻防转换清晰 |
| **蛮兽** | Rhinar, Reckless Rampage | 高风险高回报，Intimidate 有张力 |

### 9.2 核心数据模型

```
Hero {
  name: string
  class: string
  talent: string?
  life: number         // 生命值
  intellect: number    // 智力值（抽牌数）
  ability: string      // 特殊能力描述
}

Card {
  name: string
  type: "action" | "attack_reaction" | "defense_reaction" | "instant" | "weapon" | "equipment"
  subtype: string?     // "attack", "aura", "item" 等
  class: string        // 所属职业
  talent: string?      // 所需天赋
  pitch: 1 | 2 | 3    // 资源产出 (红/黄/蓝)
  cost: number         // 使用费用
  power: number?       // 攻击力
  defense: number?     // 防御值
  keywords: string[]   // 关键词列表
  text: string         // 效果描述
  rarity: "common" | "rare" | "majestic" | "legendary" | "fabled"
}

Equipment {
  name: string
  slot: "head" | "chest" | "arms" | "legs" | "off-hand"
  class: string
  defense: number
  keywords: string[]
  ability: string
}
```

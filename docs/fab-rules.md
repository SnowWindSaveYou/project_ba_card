# 《血肉之战》(Flesh and Blood) 完整规则参考

> 官方中文译名：赤魂战纪 (Flesh and Blood, 简称 FaB)
> 开发商：Legend Story Studios (新西兰奥克兰)
> 类型：1v1 交换式卡牌游戏 (TCG)

---

## 一、游戏概述

FaB 是一款 1v1 决斗式卡牌游戏，模拟两位武士之间的战斗。核心设计理念：

- **没有法力曲线**：不会像万智牌那样"卡地"
- **每张牌多用途**：同一张牌可以用来攻击、防御或支付资源，但同一张牌每回合只能选择一种用途
- **技术导向**：强调玩家决策而非运气
- **渐进优势**：通过每回合的攻防累积优势，而非猛攻速胜

---

## 二、胜利条件

- 将对手的生命值降至 **0** 即获胜
- 在限时赛制中，时间结束时生命值较高者获胜

---

## 三、游戏区域 (Zones)

| 区域 | 英文 | 说明 |
|------|------|------|
| **英雄区** | Hero Zone | 放置英雄卡，是玩家的代表 |
| **武器区** | Weapon Zone | 放置 1-2 把武器（取决于单手/双手） |
| **装备区** | Equipment Zone | 最多 4 件装备：头部、胸部、手臂、腿部 |
| **牌库** | Deck | 主牌堆 |
| **手牌** | Hand | 当前持有的牌 |
| **军械库** | Arsenal | 英雄卡下方，存放 1 张留到下回合的牌 |
| **战斗链** | Combat Chain | 攻击、防御、反应牌的解算区 |
| **资源区 (Pitch Zone)** | Pitch Zone | 用于支付费用的牌，回合结束后洗回牌库底部 |
| **弃牌堆** | Graveyard | 已使用/已解算的牌 |
| **放逐区** | Banish Zone | 被永久移除的牌 |

---

## 四、卡牌类型

### 4.1 英雄卡 (Hero)

英雄是玩家的化身，决定了：
- **生命值** (Life)：起始血量（成人版通常 20-40，闪电战年轻版较低）
- **智力值** (Intellect)：每回合结束时抽牌数量（通常为 4）
- **职业** (Class)：决定可以使用的牌池
- **天赋** (Talent)：部分英雄拥有，进一步限定牌池

### 4.2 武器卡 (Weapon)

- 游戏开始时已在场上
- 分为 **单手 (1H)** 和 **双手 (2H)** 两种
- 使用武器攻击需消耗行动点和可能的资源费用
- 战斗链关闭后武器返回武器区（不进弃牌堆）

### 4.3 装备卡 (Equipment)

- 游戏开始时已在场上
- 四个槽位功能倾向：

| 槽位 | 英文 | 功能倾向 |
|------|------|---------|
| **头部** | Head | 抽牌/牌库操控 |
| **胸部** | Chest | 资源/费用减免 |
| **手臂** | Arms | 增加伤害输出 |
| **腿部** | Legs | 行动点/Go Again |
| **副手** | Off-Hand | 额外辅助（部分职业可用） |

- 装备可以用来防御，具有防御值
- 部分装备有 **Temper** 关键词：防御后会放置 -1 标记，防御值归 0 时销毁
- 部分装备有 **Battleworn** 关键词：每次被用来防御后放置 -1 标记

### 4.4 行动卡 (Action)

主要的手牌类型，包含以下子类型：
- **攻击行动 (Attack Action)**：对对手造成伤害
- **非攻击行动 (Non-Attack Action)**：提供各种效果

每张行动卡四角信息：
```
┌─────────────────────────┐
│ [Pitch值]    [费用Cost] │
│                         │
│      卡牌名称/效果       │
│                         │
│ [攻击力Power] [防御值Def]│
└─────────────────────────┘
```

### 4.5 反应卡 (Reaction)

- **攻击反应 (Attack Reaction)**：攻击方在反应阶段使用，增加攻击力或附加效果
- **防御反应 (Defense Reaction)**：防御方在反应阶段使用，增加防御值；可能需要支付资源费用

### 4.6 即时牌 (Instant)

- 可以在任何时机使用，不消耗行动点
- 如果带有 Go Again，使用后还能获得 1 个行动点

### 4.7 其他类型

- **灵光/光环 (Aura)**：持续效果卡
- **物品 (Item)**：场上的持续道具
- **盟友 (Ally)**：独立实体，可攻击也可被攻击
- **地标 (Landmark)**：持续场地效果
- **Token**：由效果生成的标记牌

## 五、资源系统 (Pitching)

### 5.1 Pitch 值（资源产出）

每张牌左上角有 1-3 个圆点，代表该牌被"pitch"时产生的资源数量：

| 颜色条 | Pitch 值 | 资源产出 | 特点 |
|--------|----------|---------|------|
| **红色** | 1 个圆点 | 1 资源 | 攻击力/效果最强，资源产出最少 |
| **黄色** | 2 个圆点 | 2 资源 | 中庸平衡 |
| **蓝色** | 3 个圆点 | 3 资源 | 攻击力/效果最弱，资源产出最多 |

### 5.2 资源机制

- **产生时机**：仅在需要支付费用时才能 pitch 牌
- **费用位置**：每张牌右上角的数字是使用该牌所需的费用
- **资源延续**：未用完的资源在当前回合内持续存在，可用于后续行动
- **回合结束**：未使用的资源消失
- **Pitch 去向**：被 pitch 的牌放入资源区，回合结束时**以玩家选择的顺序洗回牌库底部**

### 5.3 策略意义

- 红色牌效果强但产出少 → 适合作为攻击/使用
- 蓝色牌效果弱但产出多 → 适合作为资源支付
- **Pitch 顺序**控制了牌库循环的节奏（高级策略）

## 六、回合流程

### 6.1 回合开始
- 获得 **1 个行动点** (Action Point, AP)

### 6.2 行动阶段 (Action Phase)

玩家可以执行以下操作（需消耗行动点）：

1. **打出行动卡**：消耗 1 AP + 支付卡牌费用
2. **使用武器攻击**：消耗 1 AP + 支付武器费用
3. **使用军械库中的牌**：消耗 1 AP + 支付费用
4. **使用带行动消耗的能力**：消耗 1 AP

**Go Again**：如果打出/使用的牌带有 "Go Again"，在该牌解算后获得 1 个新的行动点，可以继续行动。

### 6.3 战斗流程（见第七节详解）

### 6.4 回合结束

1. 战斗链关闭（所有攻防牌进弃牌堆，武器回武器区）
2. Pitch 区的牌洗回牌库底部
3. 可以将 **1 张手牌** 面朝下放入 **军械库** (Arsenal)
4. 抽牌至英雄智力值（通常 4 张）
5. 轮到对手回合

---

## 七、战斗流程详解 (Combat Chain)

战斗是 FaB 的核心，每次攻击构成一个"链接" (Chain Link)：

### 7.1 攻击声明 (Attack Declaration)

- 攻击方打出攻击行动卡或使用武器攻击
- 消耗 1 行动点
- 支付卡牌/武器费用（通过 pitch 手牌）
- 战斗链打开

### 7.2 防御阶段 (Defend Step)

- 防御方可以选择用 **任意数量的手牌** 和/或 **装备** 来防御
- 防御**不需要**消耗行动点或资源
- 防御**不是**"打出"牌，而是"声明防御"
- 防御必须**同时声明**所有防御用牌
- 几乎所有牌都有防御值（右下角数字），可以用来防御

### 7.3 反应阶段 (Reaction Step)

1. **攻击反应**：攻击方先使用攻击反应牌（增加攻击力/附加效果）
2. **防御反应**：防御方使用防御反应牌（增加防御值）
   - 防御反应牌可能需要支付资源费用
   - 防御反应牌解算后成为该链接的防御牌
3. 双方交替使用反应牌，直到双方都放弃

### 7.4 伤害结算 (Damage Step)

```
实际伤害 = 攻击总力量 (攻击力 + 攻击反应) - 防御总值 (防御牌 + 防御反应 + 装备防御)
```

- 如果伤害 > 0，防御方英雄**受到伤害**（扣血）
- 如果攻击造成了伤害，视为攻击 **"命中" (Hit)**
- 命中后触发攻击牌上的 **"命中效果" (On-Hit Effect)**

### 7.5 解算阶段 (Resolution Step)

- 如果攻击牌有 **Go Again**，攻击方获得 1 个行动点
- 攻击方可以继续行动（打出新的攻击，创建新的链接）
- 新攻击会创建新的 Chain Link，重复 7.1-7.5 流程

### 7.6 战斗链关闭

以下情况战斗链关闭：
- 攻击方没有行动点且不再攻击
- 攻击方打出了**非攻击行动**（关闭战斗链）
- 战斗链关闭时，所有攻防牌进弃牌堆，武器回武器区

## 八、关键词能力 (Keywords)

### 8.1 核心能力关键词

| 关键词 | 英文 | 效果 |
|--------|------|------|
| **续行** | Go Again | 解算后获得 1 个行动点；不可叠加 |
| **压制** | Dominate | 对手只能用 1 张手牌防御此攻击 |
| **威吓** | Intimidate | 随机放逐对手 1 张手牌（回合结束时返回） |
| **幻象** | Phantasm | 如果防御方用 6 攻以上的非幻象牌防御，此攻击失效 |
| **穿透** | Piercing | 即使被完全防御仍造成 1 点伤害 |
| **压倒** | Overpower | 如果攻击命中，对手弃 1 张牌 |
| **潜行** | Stealth | 从军械库打出时不能被防御反应牌防御 |
| **伏击** | Ambush | 可以在对手回合作为即时牌打出进行攻击 |

### 8.2 装备/持续效果关键词

| 关键词 | 英文 | 效果 |
|--------|------|------|
| **战损** | Battleworn | 每次用来防御后放 -1 标记 |
| **回火** | Temper | 防御后放 -1 标记，防御值为 0 时销毁 |
| **结界屏障** | Arcane Barrier X | 支付 X 资源防御 X 点奥术伤害 |
| **法术护盾** | Spellvoid X | 阻挡 X 点奥术伤害后销毁 |
| **守护** | Ward | 受到伤害时销毁此牌替代扣血 |
| **传奇** | Legendary | 构筑牌组中只能放 1 张 |
| **专精** | Specialization | 限定特定英雄才能使用 |

### 8.3 标签关键词（条件触发）

| 关键词 | 英文 | 触发条件 |
|--------|------|---------|
| **连击** | Combo | 如果本回合之前打出了特定的牌 |
| **粉碎** | Crush | 如果此攻击造成了 4 点以上伤害 |
| **回击** | Reprise | 如果防御方使用了手牌防御 |
| **涌动** | Surge | 如果本回合你已经打出了一张特定类型的牌 |
| **破裂** | Rupture | 如果此攻击造成伤害 |

### 8.4 特殊机制关键词

| 关键词 | 英文 | 效果 |
|--------|------|------|
| **增幅** | Boost | 额外费用：放逐牌库顶 1 张牌，如果是同职业牌则获得 Go Again |
| **血债** | Blood Debt | 在放逐区时，每回合结束扣 1 点血 |
| **融合** | Fusion | 从手牌中展示一张特定天赋的牌来触发效果 |
| **本质** | Essence | 资源/融合相关的天赋互动 |

## 九、伤害类型

| 类型 | 说明 |
|------|------|
| **物理伤害** | 普通攻击造成的伤害，可被防御值抵消 |
| **奥术伤害** (Arcane Damage) | 法术造成的伤害，**不能被普通防御牌抵消**，只能被 Arcane Barrier 或 Spellvoid 防御 |

## 十、赛制规则

### 10.1 闪电战 (Blitz)

| 项目 | 规则 |
|------|------|
| **英雄** | 年轻版 (Young) |
| **卡池** | 52 张（含装备和武器） |
| **牌库** | 正好 40 张 |
| **同名牌上限** | 每张独特牌 1 张 |
| **对局时长** | 约 15-25 分钟 |
| **特点** | 快节奏、血量低、适合入门 |

### 10.2 经典构筑 (Classic Constructed)

| 项目 | 规则 |
|------|------|
| **英雄** | 成人版 (Adult) |
| **卡池** | 80 张（含装备和武器） |
| **牌库** | 至少 60 张 |
| **同名牌上限** | 每张独特牌最多 3 张 |
| **对局时长** | 约 30-50 分钟 |
| **特点** | 更考验长期资源规划和回合博弈 |

### 10.3 构筑通用规则

- **同名不同色算不同牌**：红色 Wounding Blow 和蓝色 Wounding Blow 是不同的牌
- **禁止列表**：部分过强牌在特定赛制被禁
- **Living Legend**：赢得太多比赛的英雄会被"封神退休"

## 十一、游戏开局流程

1. 双方展示英雄卡
2. 随机决定先后手
3. 将武器和装备**面朝下**放置
4. 双方洗牌并切牌
5. 翻开武器和装备
6. 各抽取等同于英雄智力值的手牌（通常 4 张）
7. **先手玩家**开始第一回合（先手第一回合通常没有"先攻优势"限制）

---

## 十二、核心策略概念

### 12.1 资源管理三难题

每张牌可以做三件事之一：
1. **攻击/使用**：发挥牌面效果
2. **防御**：用防御值抵挡对手攻击
3. **Pitch**：作为资源支付费用

→ 决定每张牌的最佳用途是游戏核心决策

### 12.2 军械库管理

- 每回合结束可存 1 张牌到军械库
- 军械库中的牌下回合可以打出
- 可以设置陷阱/蓄力下回合的关键牌

### 12.3 牌库循环

- Pitch 的牌会洗回牌库底部
- 你可以**选择 pitch 牌的入库顺序**
- 控制牌库循环节奏是高级策略

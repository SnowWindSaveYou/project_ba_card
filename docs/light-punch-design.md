# 《轻拳飞扬》游戏设计文档

> 基于 Flesh and Blood (FAB) 微创新的少女格斗卡牌游戏
> 定位：1v1 格斗卡牌 · 电子版 · 中轻度竞技

---

## 一、设计目标

### 从 FAB 继承的核心

- **资源三难题**：每张牌可攻击、防御、充能三选一 —— 这是 FAB 最精华的设计，原样保留
- **战斗链（连招）**：攻防交替解算的链式结构
- **无法力曲线**：不会"卡地"，每回合都有有意义的决策
- **同牌三色**：同一招红/黄/蓝三档力度

### 微创新方向

- 护具系统从 4 槽简化为 2 槽，新增**爆衣演出**（护具破损时的视觉奖励）
- 新增格斗题材关键词（华丽连击、破绽、闪耀时刻）
- 利用电子版优势做演出、语音、教程引导

### 不做的事

- 不做实体卡（纯电子版）
- 不做开放交易市场（避免投机，走赛季通行证模式）
- 首发不做天赋系统（留给后续赛季"觉醒"扩展）

## 二、术语对照

### 原则：通用卡牌术语保持不动，仅在有明确题材价值时重命名

| FAB 术语 | 本作术语 | 是否改动 | 理由 |
|----------|---------|---------|------|
| **通用卡牌术语** | | | |
| 手牌 Hand | 手牌 | 不改 | 通用术语，所有卡牌玩家都懂 |
| 牌库 Deck | 牌库 | 不改 | 同上 |
| 弃牌堆 Graveyard | 弃牌堆 | 不改 | 同上 |
| 放逐区 Banish | 放逐区 | 不改 | FAB 术语但已足够清晰 |
| **有题材价值的改动** | | | |
| 英雄 Hero | **角色** | 改 | "英雄"暗示奇幻史诗，"角色"更中性通用 |
| 生命值 Life | **体力** | 改 | 格斗游戏标准术语 |
| 智力值 Intellect | **专注力** | 改 | 格斗语境下"脑子转得快=每回合能想到几招" |
| 职业 Class | **流派** | 改 | 核心题材身份，格斗流派 |
| 天赋 Talent | **特质** | 改 | 首发不做，留给后续赛季 |
| 装备 Equipment | **护具** | 改 | 格斗比赛就叫护具 |
| 武器 Weapon | **架势** | 改 | "起手架势"比"武器"更贴合徒手格斗 |
| 战斗链 Combat Chain | **连招** | 改 | 格斗游戏最直觉的术语 |
| Pitch（资源产出） | **充能** | 改 | "把一张牌充能→获得体能点"比"pitch"直觉 |
| 资源 Resource | **体能** | 改 | 格斗=消耗体能来出招 |
| Arsenal 军械库 | **预备区** | 改 | "蓄力/预判"太花哨，"预备"简洁准确 |
| 行动点 Action Point | **行动点** | 不改 | 够清晰 |
| Go Again | **连招** | 改 | 格斗直觉映射 |
| **卡牌类型** | | | |
| 攻击行动 Attack Action | **攻击牌** | 简化 | 去掉"行动"二字，更简洁 |
| 非攻击行动 Non-Attack Action | **辅助牌** | 改 | "非攻击行动"太绕 |
| 攻击反应 Attack Reaction | **追击牌** | 改 | 格斗语境："追加一击" |
| 防御反应 Defense Reaction | **闪避牌** | 改 | 格斗语境："闪身格挡" |
| 即时牌 Instant | **本能牌** | 改 | "本能反应"，但可考虑合并进追击/闪避 |
| 灵光/光环 Aura | **状态** | 改 | "斗志昂扬"等持续状态 |

> **总结**：牌库/手牌/弃牌堆/放逐区/行动点 → 原样保留。
> 英雄/生命/职业/装备/战斗链/Pitch → 改为格斗术语。
> 卡牌类型名称 → 简化并格斗化。

## 三、核心机制

### 3.1 资源系统（充能）

沿用 FAB 的 Pitch 机制，重命名为"充能"：

| 档位 | 颜色标识 | 充能产出 | 牌面特征 | 格斗解释 |
|------|---------|---------|---------|---------|
| 🔴 全力 | 红 | 1 体能 | 攻击力最高 | 全力出招，没余力干别的 |
| 🟡 标准 | 黄 | 2 体能 | 中庸均衡 | 标准力度 |
| 🔵 轻巧 | 蓝 | 3 体能 | 攻击力最低 | 点到为止，省体力 |

核心三难题保持不变——每张牌每回合只能选一种用途：

```
同一张牌：
  → 出招（发挥牌面攻击力/效果）
  → 防御（用牌面防御值抵挡攻击）
  → 充能（横置产出体能，支付其他牌的费用）
```

**充能去向**：充能区的牌回合结束后以玩家选择的顺序放回牌库底部（牌库循环策略保留）。

### 3.2 回合流程

与 FAB 基本一致：

```
回合开始
  └→ 获得 1 行动点
  └→ 行动阶段
      ├→ 出攻击牌（消耗 1 行动点 + 支付体能费用）→ 进入连招流程
      ├→ 出辅助牌（消耗 1 行动点 + 支付体能费用）
      ├→ 使用架势攻击（消耗 1 行动点 + 支付费用）
      └→ 使用预备区的牌（消耗 1 行动点 + 支付费用）
  └→ 回合结束
      ├→ 连招链关闭，攻防牌进弃牌堆，架势回原位
      ├→ 充能区的牌放回牌库底部
      ├→ 可将 1 张手牌面朝下放入预备区
      └→ 抽牌至专注力上限（通常 4 张）
```

### 3.3 连招流程（= FAB Combat Chain）

```
攻击方出牌（消耗行动点 + 体能费用）
  ↓
防御方选择：用任意数量的手牌 + 护具防御（不消耗行动点/体能）
  ↓
反应阶段：攻击方出追击牌 → 防御方出闪避牌 → 交替直到双方放弃
  ↓
结算：攻击力 - 防御值 = 实际伤害
  ├→ 伤害 > 0：命中，扣体力，触发命中效果
  └→ 伤害 ≤ 0：完全格挡
  ↓
攻击牌若有"连招"关键词 → 获得 1 行动点 → 可继续出牌（新的连招链接）
```

### 3.4 伤害类型

| 类型 | 说明 | 防御方式 |
|------|------|---------|
| **物理伤害** | 普通拳脚攻击 | 手牌防御 + 护具防御 |
| **内劲伤害** | 气功系角色的特殊伤害（= FAB Arcane Damage） | 只能被"运气护体"能力防御，普通牌挡不住 |

> 内劲伤害是气功流派（第二批扩展）的核心机制，首发 4 流派暂不涉及。

## 四、角色与流派

### 4.1 首发阵容（4 角色）

| 角色 | 流派 | 对标 FAB | 体力 | 专注力 | 一句话定位 |
|------|------|---------|------|--------|-----------|
| **一之濑枫** | 剑道 | Warrior (Dorinthea) | 20 | 4 | 攻防均衡的正统派，架势（竹刀）连击 |
| **夏琳** | 跆拳道 | Ninja (Katsu) | 18 | 4 | 连招王，Combo 序列丰富，腿法华丽 |
| **云柔** | 太极 | Guardian (Bravo) | 22 | 4 | 厚血厚防，反击制裁，越打越稳 |
| **铁拳小桃** | 拳击 | Brute (Rhinar) | 19 | 4 | 高爆发高风险，震慑压迫，一拳定胜负 |

### 4.2 角色差异化设计

#### 一之濑枫（剑道）

- **架势**：竹刀（双手），攻击力 3，费用 1，自带"反击"触发词
- **角色能力**：当你用架势攻击命中时，可以立即再进行一次架势攻击（额外费用 1）
- **玩法风格**：武器连击节奏型，用架势起手打出稳定伤害，手牌跟进收尾
- **流派关键词倾向**：反击 (Reprise)

#### 夏琳（跆拳道）

- **架势**：格斗站姿（徒手），攻击力 2，费用 0，自带"连招"
- **角色能力**：当你本回合打出第 3 张攻击牌时，它获得 +2 攻击力
- **玩法风格**：Combo 序列型，追求长连招触发 Combo 奖励
- **流派关键词倾向**：连击 (Combo)

#### 云柔（太极）

- **架势**：太极起手式（徒手），攻击力 4，费用 2，无连招
- **角色能力**：每回合第一次被攻击命中后，可以从弃牌堆拿回 1 张防御值 ≥ 3 的牌
- **玩法风格**：厚防反击型，先挨打积累资源再反击
- **流派关键词倾向**：重击 (Crush)

#### 铁拳小桃（拳击）

- **架势**：拳击站姿（双拳），攻击力 3，费用 0，自带"震慑"
- **角色能力**：你的攻击牌命中时，可以弃 1 张随机手牌来额外造成 2 点伤害
- **玩法风格**：高风险高回报，弃牌驱动的爆发型
- **流派关键词倾向**：震慑 (Intimidate)

### 4.3 第二批扩展角色（预留）

| 角色 | 流派 | 对标 FAB | 核心机制 |
|------|------|---------|---------|
| 灵儿 | 气功 | Wizard | 内劲伤害（不可普通格挡） |
| MIKO | 自由搏击 | Mechanologist | 道具链 + 爆气 (Boost) |
| 阿酒 | 醉拳 | Illusionist | 虚招（可被识破） |

## 五、护具系统（2 槽 + 爆衣）

### 5.1 基本结构

| 项目 | 设计 |
|------|------|
| 槽位数 | 2（上半身 / 下半身） |
| 放置时机 | 开局前选好，游戏开始即在场 |
| 防御方式 | 防御阶段可以选择使用护具抵挡（与手牌同时声明） |
| 磨损规则 | 每次用于防御后耐久 -1，防御值 = 当前耐久 |
| 破损触发 | 耐久归 0 时护具破损移除 → 触发破损效果 + 爆衣演出 |

### 5.2 磨损递减机制

与 FAB 原版的关键区别：FAB 装备防御值固定（Battleworn 只是到 0 时销毁），本作的**防御值随耐久递减**。

```
示例：加固运动背心（耐久 3）

  满耐久 ●●● → 防御 3 → 使用后耐久变 ●●○
  耐久 2  ●●○ → 防御 2 → 使用后耐久变 ●○○
  耐久 1  ●○○ → 防御 1 → 使用后耐久变 ○○○ → 破损！触发效果
```

这制造了一个递进决策：护具越早用防御值越高，但也越早耗尽。

### 5.3 爆衣演出（Armor Break）

护具破损时的电子版独占视觉演出。

**演出定位**：运动向战损，重点是打击力度感。护具碎裂后露出运动内衬/绷带/护具底层。不做过度暴露，保证全年龄段评级。

| 触发情况 | 演出等级 | 描述 |
|---------|---------|------|
| 正常磨损归零 | 普通破损 | 护具碎裂飞散短动画 |
| 被单次 ≥ 5 伤害直接打爆 | 华丽爆衣 | 慢动作 + 冲击波 + 特写 |
| 同回合双护具都被打爆 | 双重爆衣 | 稀有全屏演出 + 成就 |

### 5.4 破损效果设计原则

**失去防御的同时获得进攻补偿** —— 让破损成为"危机中的转机"。

```
护具完好 → 有额外防御，安全但平淡
    ↓ 护具破损
护具破碎 → 失去防御缓冲，但触发一次强力效果

→ 博弈：故意早用护具提前触发效果？还是多撑几回合防御？
```

## 六、关键词系统

### 6.1 从 FAB 直接继承（换名）

| 本作关键词 | FAB 原名 | 效果 |
|-----------|---------|------|
| **连招** | Go Again | 此牌解算后获得 1 行动点，可继续出牌 |
| **必杀** | Dominate | 对手只能用 1 张手牌防御此攻击 |
| **震慑** | Intimidate | 随机封印对手 1 张手牌（回合结束归还） |
| **重击** | Crush | 此攻击造成 ≥ 4 伤害时触发额外效果 |
| **反击** | Reprise | 对手用手牌防御时触发额外效果 |
| **连击** | Combo | 本回合此前打出了指定牌时触发 |
| **穿透** | Piercing | 即使被完全防御仍造成 1 点伤害 |
| **压倒** | Overpower | 命中后对手弃 1 张牌 |
| **潜行** | Stealth | 从预备区打出时不能被闪避牌防御 |
| **伏击** | Ambush | 可在对手回合作为本能牌打出 |

### 6.2 从 FAB 改造的关键词

| 本作关键词 | FAB 原名 | 改动说明 |
|-----------|---------|---------|
| **虚招** | Phantasm | 原版：被 6 攻以上非幻象牌防御则失效。改为：被任意防御值 ≥ 4 的单张牌防御则失效。简化判定条件 |
| **爆气** | Boost | 原版不变：额外放逐牌库顶 1 张，若为同流派则获得连招 |
| **疲劳** | Blood Debt | 原版：放逐区每张血债牌回合末扣 1 血。改为：放逐区每 2 张疲劳牌回合末扣 1 体力。降低惩罚烈度 |

### 6.3 本作新增关键词

| 关键词 | 效果 | 设计意图 |
|--------|------|---------|
| **华丽连击** | 本回合第 3 次及以上攻击，每次额外 +1 伤害（累加） | 奖励长连招，制造越打越猛的爽感 |
| **破绽** | 你上回合未防御任何攻击时，本回合首次攻击 +2 | 鼓励"放弃防御全力进攻"的赌博策略 |
| **闪耀时刻** | 本回合你造成的总伤害 ≥ 8 时，触发奖励效果（抽 1 牌或回复 1 体力） | 打出漂亮连招的里程碑奖励 |
| **应援** | 从放逐区拉回 1 张本流派的牌到手牌 | 资源回收，降低放逐惩罚感，适合轻松调性 |

## 七、卡牌设计

### 7.1 卡面布局

```
┌──────────────────────────────┐
│  [充能值]            [费用]  │   左上：充能值（●/●●/●●●）
│                              │   右上：出招消耗的体能
│        招式名称               │
│        ─────                  │
│   流派 · 攻击牌               │   类型行
│                              │
│   效果文字                    │
│   "连招"                     │   关键词
│                              │
│  [攻击力]            [防御值] │   左下：攻击力  右下：防御值
└──────────────────────────────┘
```

### 7.2 同招三色（同名三档力度）

同一招有红/黄/蓝三个版本，以"回旋踢"为例：

| 版本 | 充能值 | 费用 | 攻击力 | 防御值 | 名称标注 |
|------|--------|------|--------|--------|---------|
| 🔴 红 | 1 | 0 | 6 | 3 | 回旋踢 |
| 🟡 黄 | 2 | 0 | 5 | 3 | 回旋踢 |
| 🔵 蓝 | 3 | 0 | 4 | 3 | 回旋踢 |

> 三色视为不同的牌，构筑中各自计算张数上限。

### 7.3 卡牌类型总览

| 类型 | 说明 | 消耗行动点 | 常见张数占比 |
|------|------|-----------|-------------|
| **攻击牌** | 对对手造成伤害 | 是 | ~55% |
| **辅助牌** | 非伤害效果（站位调整、回气等） | 是 | ~15% |
| **追击牌** | 攻击方反应阶段使用，增加攻击力 | 否 | ~10% |
| **闪避牌** | 防御方反应阶段使用，增加防御 | 否 | ~10% |
| **本能牌** | 任何时机使用，不消耗行动点 | 否 | ~5% |
| **状态牌** | 持续效果（斗志昂扬等） | 是 | ~5% |

### 7.4 稀有度

| 等级 | 对标 FAB | 说明 |
|------|---------|------|
| **基础** | Common | 基本功招式 |
| **进阶** | Rare | 需要苦练的技巧 |
| **精英** | Majestic | 看家本领 |
| **奥义** | Legendary | 传说绝招，构筑中限 1 张 |

### 7.5 数据模型

```
Character {
  name: string          -- "一之濑枫"
  style: string         -- "剑道"
  hp: number            -- 体力值
  focus: number         -- 专注力（每回合抽牌数）
  ability: string       -- 角色特殊能力描述
}

Stance {                -- 架势（= FAB Weapon）
  name: string          -- "竹刀"
  hands: 1 | 2          -- 单手/双手
  style: string         -- 所属流派
  power: number         -- 攻击力
  cost: number          -- 使用费用
  keywords: string[]    -- 关键词
}

Card {
```

## 八、数值框架

### 8.1 基础参数

| 参数 | 值 | 对比 FAB 闪电战 | 理由 |
|------|-----|---------------|------|
| 体力值 | 18-22（按角色） | 20 | 基本一致 |
| 专注力 | 4 | 4 | 不变，4 张手牌的决策量刚好 |
| 牌库张数 | 30 | 40 | 缩减，电子版节奏更快 |
| 同名牌上限 | 每张 2 | 每张 1 | 允许少量重复，降低收集压力 |
| 预备区容量 | 1 张 | 1 张 | 不变 |
| 护具槽数 | 2 | 4 | 简化 |

### 8.2 攻防数值区间

| 项目 | 常见范围 | 极端值 |
|------|---------|--------|
| 攻击牌攻击力 | 3-7 | 红牌奥义可达 9-10 |
| 攻击牌防御值 | 2-3 | 少数为 1 或 4 |
| 追击牌加攻 | +2~+3 | 极端 +4 |
| 闪避牌加防 | +2~+4 | — |
| 架势攻击力 | 2-4 | — |
| 护具耐久 | 1-4 | — |
| 牌的体能费用 | 0-3 | — |

### 8.3 一回合伤害期望

| 进攻模式 | 预期总攻击力 | 说明 |
|---------|------------|------|
| 单次重击 | 5-7 | 出 1 张红牌大招 |
| 两连招 | 8-11 | 架势起手 + 1 张攻击牌 |
| 三连招 | 11-15 | 3 张攻击牌串联 |
| 完美连招（罕见） | 16+ | 4 张全攻 + 追击牌 |

对应防御方 4 张手牌全防约 10-12 防御值 + 护具 2-4 = 总共 12-16。与进攻数值基本匹配，保证多数回合有少量伤害穿透。
